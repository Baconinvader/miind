MIIND LIF GeomAlgorithm Example 3

A legacy 1D version of the MIIND population density technique for a population of LIF neurons (GeomAlgorithm). twopop defines a fully connected E-I network of populations.

Requires ROOT, MIIND built and installed from source, Linux MIIND Standalone

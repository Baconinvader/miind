MIIND Tutorial from CNS2019

This tutorial was given in person at the CNS 2019 conference in Barcelona. It was not designed for use with the pip version of MIIND, only the standalone version which is built from source. 
For a similar introduction to MIIND v1.0.11 which is available through pip, see our publications. 

Requires ROOT, MIIND built and installed from source, Linux MIIND Standalone
